# coc-apollo-graphql

coc.nvim extension for Apollo GraphQL

## Install

`:CocInstall coc-apollo-graphql`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-apollo-graphql-keymap)`

## Lists

`:CocList demo_list`
